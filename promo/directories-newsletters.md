# Directory & Newsletter Submissions

Copy/paste blurbs for tool directories and newsletter tips. Most are web forms —
paste the matching blurb, attach `promo/images/report-preview.png` where a logo
or screenshot is asked for.

## Reusable blurbs

**One-liner (≤80 chars):**
```
Open-source CLI that turns a company name into an evidence-labeled business model report.
```

**Short (≤160 chars):**
```
OpenBusiness is a free, open-source CLI. Give it a company + domain; it runs LangGraph analyst agents over public evidence and writes a business-model report that tags every claim verified, inferred, or missing.
```

**Medium (≤320 chars):**
```
OpenBusiness is an open-source (MIT) Python CLI for business-model research. Given a company name and domain, it collects public evidence (Tavily, Firecrawl, SEC EDGAR) and runs 9 LangGraph analyst agents — JTBD, GTM, unit economics, moat, canvas, and an assumption stress test — producing a Markdown report where every claim is tagged verified, inferred, or missing. Providers: OpenAI, Anthropic, DeepSeek. Reports cost ~$0.10–$0.40.
```

**Category / tags:** AI Agents · Developer Tools · Market Research · Business Intelligence · Open Source · CLI
**Pricing:** Free / Open Source (bring your own LLM key)
**Links:** repo https://github.com/wanikua/OpenBusiness · PyPI https://pypi.org/project/openbusiness/

## Newsletters (tip / submission)

Lead with the concrete hook, not the feature list. Most accept tips by email or
a form.

**Pitch (paste into the tip box / email body):**
```
Hook: an open-source CLI ran Notion and modeled LTV/CAC at 1.6x — then showed that one churn assumption (4% → 6%) flips it to 1.07x, or (→ 2%) to 3.2x.

OpenBusiness turns a company name into a business-model report from public evidence, and tags every claim verified / inferred / missing instead of inventing private-company numbers. 9 LangGraph analyst agents, MIT, runs on OpenAI/Anthropic/DeepSeek.

Repo: https://github.com/wanikua/OpenBusiness
A full sample report (Notion) is in the examples/ folder.
```
